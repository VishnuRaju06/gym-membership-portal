# Gym Membership Portal

A web-based application designed to simplify the gym admission and membership process. 
The system allows users to apply for gym membership online, choose fitness plans, and manage their profiles, 
while administrators can efficiently manage admissions, members, and gym plans through a dedicated dashboard.
