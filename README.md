# Gym Membership Portal

A web-based application designed to simplify the gym admission and membership process. 
The system allows users to apply for gym membership online, choose fitness plans, and manage their profiles, 
while administrators can efficiently manage admissions, members, and gym plans through a dedicated dashboard.

---

## Features

### User Features
- Online gym membership registration
- Selection of membership plans
- User profile management
- Admission request submission
- Responsive UI for desktop and mobile

### Admin Features
- View and manage member applications
- Approve or reject admissions
- Manage membership plans
- View registered users and membership status
- Dashboard for gym operations
